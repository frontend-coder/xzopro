HTML5 шаблон XZOPRO

Выполняю ручную верстку сложного HTML5 шаблона XZOPRO для создания блога или сайта-визитки по теме IT.

Макетом предусмотрены такие оригинальные страницы:

главная страница страницы
страница о нас
страница услуги
страница проекты или портфолио
раздел блога
страница контакты

На основе макета создаются другие внутрение страницы сайта: поиска, тегов, обычные внутрение страницы и так далее.

После завершения ручной верстки всех страниц буду выполнять посадку HTML-посадку на WordPress. В процессе посадки буду использовать специализированный фреймворки создания административной панели сайта/посадки.
